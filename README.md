# Sensores via sockets

Projeto 1 da disciplina de Redes de Computadores do Instituto de Ciências Matemáticas e de Computação (ICMC-USP).

## Requisitos

```
gcc
OpenGL
SOIL
```

## Compilação e execução
```
$ make
```
### Servidor
```
$ ./server <porta>
```
### Clientes
```
$ ./client localhost <porta>
```
